# TicketMastery Dashboard Frontend (CSV Prototype)

This is a standalone frontend prototype for your dashboard using local CSV data.



## Run

1. Open `dashboard/index.html` in a browser.
2. Load `merged_tickets_assets.csv` into the interface.

## Map

1. Install Pandas, PyDeck, and Streamlit
2. Load 'TICKETS_WITH_COORDS' into 'map_app.py'
3. Run 'streamlit run map_app.py' in the terminal
